第四章　唐诗弹窗注释示例

第一節　何謂弹窗注释

弹窗注释（Popup Annotation）是电子书中常见的交互形式：正文内以小号数字标记，读者点击后弹出注释浮层，阅读完毕后关闭返回。这种形式在 Apple Books、Google Play Books 等现代阅读器中均有原生支持。

本範本提供两种实现方式：

  • 方式一：EPUB3 原生弹窗 — 使用 epub:type="noteref" 与 epub:type="footnote"，Apple Books 等自动渲染为原生弹窗，无需额外 CSS
  • 方式二：CSS :target 弹窗 — 纯 CSS 实现，通过 URL hash 激活，兼容 Readium、Calibre Viewer 等

此处的「弹窗注释」指的是读者点击正文中的标记（如 [1]）后，在页面上方或侧边弹出的浮层窗口，可阅读注释内容后关闭返回。这种交互形式广泛存在于现代 EPUB 阅读器中。

第二節　《靜夜思》全文弹窗注释

以下为李白《静夜思》全文，配合五处弹窗注释。点击每句诗后的上标数字可查看注释弹窗：

靜夜思

李白

床前明月光， ①

疑是地上霜。 ②

舉頭望明月， ③

低頭思故鄉。 ④

注　① ✕

床：此字歷來有「窗」「床榻」「井欄」三說。宋代人注《靜夜思》多作「床前看月光」，可知宋人理解的「床」為「窗」之義，與後文「舉頭望明月」相呼應。

月：月光。詩題「靜夜思」，首句即以月光破題，靜中有動。

↑ 返回正文
注　② ✕

霜：地面凝結的薄冰。此句以「霜」比月光，極言月色之清冷皎潔。李白以視覺錯覺入詩，月光映地，如霜滿地，營造出一片清寒寂靜的秋夜氛圍。

↑ 返回正文
注　③ ✕

舉頭：抬頭。明月當空，詩人抬頭凝望，情感由外境轉入內心。明月在中國詩歌傳統中象徵團圓與思念，是古詩中最常見的意象之一。

↑ 返回正文
注　④ ✕

故鄉：詩人故里。詩到此句意境突轉，由景入情。詩人由望月而懷人，由月光之清冷而觸發思鄉之情。短短二十字，情景交融，為千古傳誦之名篇。

此詩版本異文較多通行本「舉頭望明月，低頭思故鄉」，宋蜀本《李太白文集》作「擧頭望山月，低頭思故鄉」。

第三節　实现原理

弹窗的核心机制是 CSS :target 伪类。当 URL 中的 hash（如 #fn1-source）与元素的 id 匹配时，:target 生效，弹窗显示。

<!-- 正文中的注释标记 -->
<a class="noteref" href="#fn1-source">①</a>

<!-- 弹窗定义（id 必须与 href hash 一致） -->
<aside id="fn1-source" class="annotation-popup">
    <div class="popup-header">
        <span class="popup-num">注　①</span>
        <a class="popup-close" href="#">✕</a>
    </div>
    <p>注释内容...</p>
    <a class="popup-back" href="#">↑ 返回正文</a>
</aside>
/* CSS 核心：:target 激活弹窗 */
.annotation-popup {
    display: none;
}
.annotation-popup:target {
    display: block;  /* 激活时显示 */
}

竖排中的弹窗定位

在 writing-mode: vertical-rl 中，弹窗需要特殊定位：

  • 弹窗置于屏幕左侧（即竖排的内联末端方向）
  • 弹窗内容使用 writing-mode: horizontal-tb，方便横向阅读长注释
  • 宽度设为 14em（窄而高，符合竖排空间利用）

完整 CSS 样式请参见 styles/vertical.css 的 Section 18。
