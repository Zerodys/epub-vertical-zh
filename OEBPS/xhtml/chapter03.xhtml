第三章　雙行夾注

第一節　何謂雙行夾注

雙行夾注，又稱側注、邊注，是中國古籍中最常見的註疏形式。其法：大字正文居右，小字注釋居左，兩者橫向對齊，如影隨形。此制始於宋代，盛於明清，凡儒家經典無不有此體例。

朱熹《四書章句集注》即為雙行夾注之典範：

孔氏之遺書也。初釋經一章，闡明明德之義。 大學之道，在明明德，在親民，在止於至善。
明，明顯也。復其所得於天之光明本性也。 明明德。
親，近也。程子曰：「親民」當作「新民」。 親民。
止，居也。事物皆有倫理，人當止於其善。 止於至善。

如上所示，每一夾注單元由兩部分構成：左側為朱熹之注，右側為經文原文。注字約佔原文百分之五十五的大小，以紅色區分之，視覺層次分明。

第二節　CSS 實現方式

本範本採用 flex 佈局配合 writing-mode: vertical-rl 實現夾注：

<div class="vertical-note">
  <span class="note-text">注釋文字（小字紅色）</span>
  <span class="main-text">正文文字（大字黑色）</span>
</div>

核心原理：

  • display: flex; flex-direction: row — 將注與正文橫向並排
  • justify-content: flex-start — 注釋靠左，正文填滿右側
  • 注與正文各自設定 writing-mode: vertical-rl — 兩者均為豎排
  • font-size: 0.55em — 注釋縮小至原文約五成
  • color: #c0392b — 紅色（接近硃砂色）區分注釋

第三節　詩經示例

《詩經》鄭玄箋亦用此體：

關睢之鳥摯而有別，后妃說樂君子。 關關睢鳩，在河之洲。
窈窕，幽閒也。淑，善也。逑，匹配也。 窈窕淑女，君子好逑。

第四節　Ruby 夾注混合

夾注中亦可加入 ruby 注音，適用於難字注讀：

詩ㄕ者，志之所ㄙㄨㄛˇ之也。 詩言志。

第五節　純段落正文

此節為不使用夾注的純段落正文，適用於一般敘述性文字。首行縮排兩字，標準行距一點八倍，兩端對齊。此段落不受夾注樣式影響，維持正常閱讀節奏。

再續一段，以展示長段文字在豎排中的呈現效果。中文豎排之妙，在於文字如流水般自上而下、自右而左，讀者目光隨字而行，自然形成韻律之感。

第六節　Kindle 兼容性說明

本範本在 Apple Books、Google Play Books、Calibre Viewer 等閱讀器中可完整呈現豎排效果。若目標設備為 Amazon Kindle，需注意以下事項：

Calibre 轉換時的已知問題

Calibre 在轉換 EPUB 至 AZW3 時，會刪除 -webkit-writing-mode 和 -epub-writing-mode 屬性（Calibre Bug #1862447）。解決方式是透過 Extra CSS 重新注入宣告：

.vrtl { writing-mode: vertical-rl; }
.hltr { writing-mode: horizontal-tb; }

Kindle Enhanced Typesetting

Amazon Kindle 的 Enhanced Typesetting 模式支援 writing-mode: vertical-rl 和 -webkit-text-orientation（含 upright）。在 KDP 上傳書籍後，此功能會自動啟用（若書籍通過檢查）。

Kindle 雙行夾注 Fallback

Kindle 上的 flex 布局支援不穩定，若雙行夾注顯示異常，可使用 float 方式的 fallback 類：

<div class="vertical-note">
  <span class="note-text vn-note-kindle">注：小字紅色，浮至右側</span>
  <span class="main-text vn-main-kindle">大學之道，在明明德。</span>
</div>

完整 Kindle Extra CSS 片段請參見 styles/vertical.css 的 Section 17，以及專案 README.md 中的「Kindle 轉換指南」。
